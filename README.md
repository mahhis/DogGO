ОПИСАНИЕ ПРОЕКТА
Причины инициации проекта:
Получение опыта разработки.
Содержание проекта: 
Проект “DogGo” представляет собой веб-сервис для выгула животных.
Сервис предназначен для людей, у которых не хватает времени чтобы выгулять своих питомцев.
Веб-сайт будет иметь удобный и интуитивный пользовательский интерфейс. 
Размещение новых объявлений не занимает много времени.
Пользователям доступен удобный фильтр для быстрого и точного поиска желаемой информации.  
Сервис предоставляет информацию о том где требуется выгул и какую стоимость будет иметь данная услуга.
